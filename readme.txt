Geocache Stat Bar widget

== Description ==

= FRANCAIS =

Nous avons construit un simple petit widget de sorte que vous pouvez afficher Geocache Stat Bar comme un widget dans la barre latérale de votre thème WordPress!

Tout ce que vous avez à faire est de télécharger le widget, installez-le dans votre dossier de plug-ins, l'ajouter à votre barre latérale et de modifier les préférences. Entrez votre nom d'utilisateur, GUID, sélectionnez l'image et entrez votre message.

Pour obtenir votre GUID, connectez-vous a votre compte, allez sur "Mon compte", puis cliquez sur "View My Stat Bar" (près du bas sur le côté droit). Vous verrez un code généré dans la zone de texte.

Vous devez copier (coller) tout ce qui suit "guid =" et avant la dernière citation dans la ligne 1, comme tant - guid = db5629ca-06d2-4f95-8c72-bdb425934722 "Nous savons que cela est un peu archaïque en ce moment et est l'une des choses que nous espérons remédier avant la version finale.

En outre, la géocache Stat Bar est de 200 pixels de large, alors assurez-vous espace encadré est mis en place pour gérer la largeur.

= ENGLISH =

We’ve built a simple little widget so you can display you Geocache Stat Bar as a widget in your WordPress theme’s sidebar!

All you have to do is download the widget, install it in your plug-ins folder, add it to your side bar and edit the preferences.  Enter you username, GUID, select image and enter your message.

To obtain your GUID, log in to you Geocache account, go to “My Account,” then click “View My Stat Bar” (near the bottom on the right side under “Account Options”).  You’ll see some generated code in the text area.

What you want to copy (and paste) is everything after “guid=” and before the last quote in line 1, like so - guid=db5629ca-06d2-4f95-8c72-bdb425934722” We know this is a bit archaic right now and is one of the things we hope to remedy before version 1.

Also, the Geocache Stat Bar comes 200 pixels wide, so be sure you sidebar area is set up to handle the width.

== Installation ==

1. Upload `geocache-statbar.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Obtain your GUID: To obtain your GUID, log in to you Geocache account, go to “My Account,” then click “View My Stat Bar” (near the bottom on the right side under “Account Options”).
